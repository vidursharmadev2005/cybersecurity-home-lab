# Cybersecurity Home Lab

## Overview

This project documents the creation of my personal cybersecurity home lab built using Ubuntu Linux in VirtualBox.

The objective of this lab is to gain practical experience with Linux administration, networking, vulnerability assessment, and Security Operations Center (SOC) concepts through hands-on experimentation.

The repository serves as both a learning journal and a technical reference for the tools, configurations, and exercises completed throughout the project.

---

## Objectives

- Build a Linux-based cybersecurity lab
- Practice Linux system administration
- Understand networking fundamentals
- Perform network reconnaissance using Nmap
- Deploy and configure Wazuh SIEM
- Analyze security logs and alerts
- Document the complete setup process

---

## Technologies

- Ubuntu Linux
- VirtualBox
- Git & GitHub
- Nmap
- Wazuh SIEM
