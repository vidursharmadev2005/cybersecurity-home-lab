# Wazuh SIEM Installation

## Objective

Deploy a Security Information and Event Management (SIEM) platform to monitor system activity, collect logs, detect security events, and gain hands-on experience with SOC workflows.

---

## What is Wazuh?

Wazuh is an open-source SIEM and XDR platform used to monitor endpoints, analyze security events, detect threats, and assist Security Operations Center (SOC) analysts in investigating incidents.
