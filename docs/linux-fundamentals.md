# Linux Fundamentals

## Objective

Prepare the Ubuntu virtual machine for cybersecurity testing by updating packages and installing essential utilities.

## Tools Installed

- Git
- Curl
- Wget
- Net-tools
- Tree
- Htop
- Vim

## Skills Learned

- Package management using APT
- Linux filesystem navigation
- System monitoring
- Command-line basics

## Commands Used

```bash
sudo apt update
sudo apt upgrade -y
sudo apt install git curl wget net-tools unzip tree htop vim software-properties-common apt-transport-https ca-certificates gnupg lsb-release -y
```
