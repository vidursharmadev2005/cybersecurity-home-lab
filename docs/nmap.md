# Nmap Fundamentals

## Objective

Learn how to discover hosts, identify open ports, detect services, and understand basic network reconnaissance.

## Exercises Performed

- Verified Nmap installation
- Scanned localhost
- Scanned the VM's IP address
- Detected services using `-sV`
- Attempted OS detection using `-O`
- Created a temporary HTTP server on port 8000
- Verified the open port with Nmap

## Key Learnings

- Closed ports indicate no listening services.
- Open ports expose running services that can be investigated.
- Service detection helps identify applications.
- Nmap is one of the most widely used reconnaissance tools in cybersecurity.
